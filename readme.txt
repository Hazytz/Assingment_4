
---------------------- Compilação ----------------------

Utilize um comando "makefile run" para compilar e executar o código.


A mudança do modelo não foi implementada de forma externa ao código, então é necessario mudar manualmente o filename do arquivo, ele se encontra no topo da main.cpp.

---------------------- Controles ----------------------

Digide 1 ou 0 ao abrir o executavel para selecionar entre OpenGL ou Closed2GL

Todos no menu interno.

MOUSE:
- Botão esquerdo pressionado + movimento: rotaciona a câmera (FreeCam)
- Scroll do mouse: aproxima/afasta (modo LockedCam)
